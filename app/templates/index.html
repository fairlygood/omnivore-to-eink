Omnivore to PDF

A Fairly Good Web App

This application will fetch a bunch of articles from Omnivore and generate a PDF in a size suitable for large-format e-readers, such as the Kindle Scribe or Remarkable.

Settings
Enter your API key in settings to get started.

Note: We'll only return a maximum of ten articles because generating the PDF can be resource intensive, especially if the documents are long or have lots of images. It also results in smaller file sizes (and less bandwidth).

Enter a label (optional) to return only articles with that label applied, and choose whether to return the newest or oldest articles.
